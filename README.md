# WMX3 Simulator API - Digital I/O Simulation
WMX3のシミュレータ機能を用いて、実機（ハードウェア）がない環境でデジタルI/Oの挙動をシミュレート・テストするための解説およびサンプルコードです。

📌 概要
このプロジェクトでは、WMX3 APIを使用して以下の操作をシミュレーション環境で行う方法を解説します。

シミュレータモードでのデバイス作成

仮想入力（Input）信号のセット（外部機器からの入力を模倣）
仮想出力（Output）信号の読み取り（ロジックの結果を確認）


🔧 開発環境 (Requirements)
OS:Windows

WMX3.6

Visual Studio 2019



📊 動作確認方法
WMX3 Console を起動します。

I/O Monitor を開きます。

APIから送信した信号が、コンソールの画面上で変化することを確認してください。
